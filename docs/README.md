# Capital Gains Landing Page

דף נחיתה סטטי עבור Capital Gains.

## מבנה

- `index.html` — עמוד נחיתה בעברית, RTL, מוכן לפרסום.
- `styles.css` — עיצוב Premium פיננסי, רספונסיבי, Navy/Gold לפי שפת המותג.
- `assets/capital-gains-logo.svg` — לוגו וקטורי לשימוש בכותרת וברקע.
- `assets/favicon.svg` — favicon.

## שילוב הלוגו ברקע

הלוגו משולב במספר אזורים בעמוד עם שקיפות משתנה:

- Hero כהה: Watermark גדול ועדין.
- אזורי תוכן לבנים: לוגו ברקע בשקיפות נמוכה.
- אזור פרטיות ו־CTA: לוגו מודגש מעט יותר אך עדיין לא פוגע בקריאות.

## פרסום דרך GitHub Pages

העמוד נמצא תחת `docs/`, כדי שיהיה קל להפעיל GitHub Pages על:

`Settings → Pages → Deploy from branch → main / docs`

כרגע הקישורים ל־Microsoft Store ול־Google Play הם placeholders. לאחר העלייה לחנויות צריך להחליף אותם בכתובות האמיתיות.

## נקודות לעדכון בהמשך

1. להחליף את כפתור Microsoft Store בקישור האמיתי.
2. להחליף או להסתיר את Google Play עד שיש מסלול Android בשל.
3. להוסיף צילום מסך אמיתי של האפליקציה במקום ה־mock dashboard.
4. להוסיף Privacy / Terms אם העמוד ישמש כעמוד Store רשמי.
