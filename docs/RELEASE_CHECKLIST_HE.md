# צ'קליסט שחרור להפצה - Capital Gains

מטרת המסמך היא לאפשר שחרור מסודר של גרסת Desktop, בלי לשכוח בדיקות תפעוליות, UX, קבצים נלווים, או נקודות רגישות סביב Google Sign-In.

הצ'קליסט הזה מיועד לעבודה לפני:

- יצירת `EXE`
- אריזת release ללקוח
- מסירה לבדיקה
- העלאה עתידית ל-Microsoft Store
